# Data Types and Loops

Introductory Python coursework focused on collections, loops, string slicing, and basic calculations.

## Notebook

`hw1 loops.ipynb` contains four exercises:

1. Return the middle character or characters from a string.
2. Read numbers until zero and calculate their sum.
3. Sort and pair two name lists when their lengths match.
4. Convert average country temperatures from Fahrenheit to Celsius.

## Skills practiced

- Strings and slicing
- Lists, sorting, and `zip`
- `while` and `for` loops
- Nested data structures
- Aggregation and unit conversion

## Run locally

```bash
python -m pip install notebook
jupyter notebook
```

Then open `hw1 loops.ipynb` and run the cells in order.
